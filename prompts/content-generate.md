# 内容文案生成 Prompt

## 任务

根据用户选择的发布载体、内容方向和选题，生成一条可直接发布的内容文案。

## 输入

```json
{
  "publishChannel": "短视频/口播 | 小红书/图文 | 公众号/文章 | 头条号/文章 | 朋友圈/动态",
  "contentGoal": "...",
  "selectedDirection": "...",
  "selectedTopic": "...",
  "storyAssets": [],
  "preferredTone": "真实、自然、像人话"
}
```

## 输出规则

### 短视频 / 口播

```markdown
# 标题
...

# 开头 3 秒
...

# 正文口播
...

# 结尾引导
...

# 拍摄提示
...
```

### 小红书 / 图文

```markdown
# 标题
...

# 封面文案
...

# 正文
...

# 结尾互动
...

# 话题标签
...
```

### 公众号 / 头条号

```markdown
# 标题
...

# 开头
...

# 正文结构
...

# 正文内容
...

# 结尾观点 / 互动
...
```

### 朋友圈

```markdown
# 正文文案
...

# 配图建议
...

# 轻互动结尾
...
```

## 风格原则

- 像一个真实的人说出来
- 不要太官方
- 不要太营销
- 不要承诺涨粉或快速变现
- 朋友圈不要强 CTA
