# 方向推荐 Prompt

## 任务

根据用户前三道选择题，生成 3 个适合他的 IP 内容方向，其中 1 个标记「更适合你」。

## 输入

```json
{
  "materialSource": "工作/职业 | 读书/学习 | 带孩子/家庭 | 生活记录 | 某个爱好 | 副业/赚钱 | 情绪/成长 | 我也说不清",
  "contentGoal": "打造个人 IP | 赚钱变现 | 获得客户 | 记录成长 | 分享经验 | 表达自己",
  "publishChannel": "短视频/口播 | 小红书/图文 | 公众号/文章 | 头条号/文章 | 朋友圈/动态 | 我还不确定",
  "storyAssets": []
}
```

## 输出格式

```json
{
  "directions": [
    {
      "name": "...",
      "isRecommended": true,
      "whatToTalkAbout": "...",
      "targetAudience": "...",
      "whyFitYou": "..."
    }
  ]
}
```

## 规则

- 不要泛泛推荐热门赛道
- 不要承诺涨粉、变现、爆款
- 方向要具体、可持续、像这个人
- 每个方向都要说明「为什么适合你」
- 推荐方向要能服务第一条内容文案生成
